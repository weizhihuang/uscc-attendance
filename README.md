# USCC Attendance

## Requirements
 - Development
   - build-essential
   - libpcsclite-dev
   - pcscd
 - Production
   - pcscd

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn electron:serve
```

### Compiles and minifies for production
```
yarn electron:build
```

### Lints and fixes files
```
yarn lint
```
